 Supermarket Sales Analysis

Advanced Python Project (First semester 2025/2026)

 Overview
This project performs a comprehensive analysis of the Supermarket Sales Dataset. The goal is to explore sales trends, understand customer demographics, and identify high-performing branches and product lines using Python.

 Structure
- `Supermarket_Sales_Analysis.ipynb`: The main Jupyter Notebook containing the code, analysis, and visualizations.
- `Report.md`: A summary report of the key findings and answers to the project's analytical questions.
- `SuperMarket Analysis.csv`: The dataset used for this analysis.
- `README.md`: This file.

 Requirements
To run this project, you need the following Python libraries installed:
- pandas
- numpy
- matplotlib
- seaborn
- jupyter (to run the notebook)

 How to Run
1.  Open the terminal in this directory.
2.  Run `jupyter notebook` to launch the Jupyter interface.
3.  Open `Supermarket_Sales_Analysis.ipynb` and execute the cells.
